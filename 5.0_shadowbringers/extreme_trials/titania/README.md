# The Dancing Plague (Extreme)

## English
```
【Mist Runes】
　H1　　MT　　H2
　　D1 　▲　 D2
　D3　　OT　　D4
【Flame Runes】
MTH1　　OTH2
D1D3　　D2D4
　※Rotate clockwise if necessary
【Chain of Brambles】
　North：Tanks+Healers　South：DPS
【Midsummer Night's Dream】
　OT→D1→D2→D3→D4→H1
【Adds phase】
　Order:　East→North→West
　(MT takes North, OT takes West)
```

　H1　　MT　　H2
