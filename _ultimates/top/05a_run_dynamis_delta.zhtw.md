# 代碼：潛能量 (德爾塔)

P5 總共分為 3 個運動會階段 - 德爾塔、西格瑪、歐米茄.
每個運動會都有一連串的機制，並會在結尾時觸發「你好，世界」的 Debuff 時給予玩家們一定層數的 Dynamis，該 Buff 為通關最終階段的必要機制。

PF 在代碼：潛能量(德爾塔) 所使用的攻略法為 "Kinda Awk"，其特徵為運動會期間把相同的連線配對在一起。

## 『你好，世界』

這裡的『你好，世界』跟P3不一樣，是新的弱體效果，如何適當安排站位並處理潛能量的分配貫穿了P5的主軸。
「德爾塔」與後續兩個運動會「西格瑪、歐米茄」加總起來總共會有 4 組 「你好，世界」需要處理。而團隊的最終目標為**所有隊員在進入 P6 時皆擁有 3 層潛能量。**

## 太陽射線

整個 P5 總共有四次「太陽射線」。

「太陽射線」會對當前最高仇恨進行兩下爆坦級別的傷害(基礎傷害約為坦克血量
的三倍)，並附帶魔法易傷，可透過換坦改變第二下的目標。

換句話說，該機制強制隊伍進行換坦或是使用無敵。

這四次「太陽射線」有兩次要使用無敵，而另外兩次會使用減傷搭配換坦。而在哪兩次使用無敵則與隊伍職業構成有一定關係：

  <p>若是隊伍成員有「持續15秒，且CD90秒的減傷」(像是坦克群減、遠敏減傷)，可以在太陽射線的第二次攻擊判定後立刻開啟，這樣就可以在下一次的『代碼：潛能量』判定時CD結束。</p>
  <p>而治療的團隊20秒減傷，若是釋放時機抓的好的話(約第一下死刑讀條至一半時)，可以同時覆蓋『太陽射線』與『潛能釋放』。</p>

## 運行：潛能量 (德爾塔)

運行：潛能量(德爾塔)，簡稱「P5一運」，可以拆解成三個部分：

1. [拳頭碰撞處理](#1-拳頭碰撞)
2. [探測式檢知波動砲與盾擊+能量投射](#2-探測式檢知波動砲與盾擊能量投射)
3. [『你好，世界』](#3你好世界)

## 1. 拳頭碰撞

熟悉的近/遠線補丁再次出現，隊伍成員會被分成兩組：

- 4位玩家會獲得近線。
- 另外4位玩家會獲得遠線。
  - “你好，近處世界” 與 “你好，遠處世界” 則會從線組的玩家隨機各選一位附加Debuff效果。

## 2. 探測式檢知波動砲與盾擊+能量投射

誘導完拳頭的雷射後，玩家依舊會照身上的近/遠線分成兩組，而每組人要做的事情也不同：

- 近線組要去**接檢知砲**攻擊。
- 遠線組要去處理**盾擊後的分攤**傷害『能量投射』。

<details markdown=block>
<summary>
  <b>[太陽射線]</b> 如果隊伍沒有戰士，為什麼就不考慮無敵與減傷輪流交替使用？難道這樣安排不會比較輕鬆嘛？
</summary>
<table>
  <tr>
    <td>
      <p>這是為了P6預先做的減傷安排。</p>
      <p>你也可以無敵與減傷輪流開，但是P6的減傷使用就會出問題。</p>
      <p>首先，只有兩種坦克可以在P5的『代碼：潛能量(歐米茄)』時使用無敵 並且在P6波動砲時轉的回來──暗黑騎士與戰士， 所以如果坦克組成為「絕槍戰士+騎士」時，一定得要前面兩次太陽射線使用無敵才能夠應對 後面的減傷處理。</p>
      <p>這也代表，當隊伍沒有戰士時，能取捨優先級的剩餘三位坦克： 暗黑騎士+(騎士或決槍戰士)，那我們來探討一下「行屍走肉」應該使用在第二次還是第三次的太陽射線：</p>
      <p>如果暗黑騎士決定把行屍走肉使用在第三次太陽射線，那下一次CD轉好時會出現在最後一次的對坦大傷(宇宙龍炎衝)，而下面則是較為建議的減傷安排：</p>
      <ul>
        <li>[~0:00] 宇宙龍炎衝 #1 → 鐵壁</li>
        <li>[~0:46] 波動砲 #1 → 暗影牆</li>
        <li>[~1:20] 波動砲 #2 → 鐵壁</li>
        <li>[~1:54] 宇宙龍炎衝 #2 → 行屍走肉</li>
      </ul>
      <p>然而，這種減傷安排會出現以下問題：</p>
      <ul>
        <li>龍炎衝的對坦傷害大約是280K，而波動砲的前兩位(對坦)傷害是370K，以致於第二次波動砲時，暗黑騎士除了鐵壁之外， 還會需要繳交額外減傷手段。</li>
        <li>鐵壁不可能在宇宙龍炎衝 #1 至 波動砲 #2之間轉好CD。 除非你能在第一次龍炎衝時剛好讓鐵壁持續時間結束， 這樣就會變成減傷時間要抓很緊湊。</li>
        <li>而且即便有成功抓到時間，要額外繳交的減傷資源也可能讓團隊減傷安排變的更為嚴苛。 所以會建議把減傷安排在要求相對不是嚴苛的P5 會讓治療更容易能關注坦克與團隊的狀況。</li>
      </ul>
      <p>讓P5的太陽射線不要輪流開無敵，你在P6的減傷分配會更加容易且不會壓力太大：</p>
      <ul>
        <li>[~0:00] 宇宙龍炎衝 #1 → 鐵壁</li>
        <li>[~0:46] 波動砲 #1 → ST 無敵技能, MT 30%</li>
        <li>[~1:20] 波動砲 #2 → MT 無敵技能, ST 30%</li>
        <li>[~1:54] 宇宙龍炎衝 #2 → 鐵壁</li>
      </ul>
      <p>然而，這樣的減傷安排對有戰士的隊伍而言較不適用，因為戰士是唯一一個坦克可以在『代碼：潛能量(歐米茄)』時使用無敵技能，<b>且可以在第一次波動砲前就能把CD轉回來的坦克。</b></p>
    </td>
  </tr>
</table>
</details>
